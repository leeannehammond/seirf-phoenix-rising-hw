What is the difference between a client and a server?

A server is a program that waits for incoming requests from the client.
A client is also a program that sends the requests to the server to connect to perform actions. 

What is the request-response cycle?

The request-response cycle starts with a client sending a request to a server. 
The server handles the request and sends a response back to the client. 
An HTTP message has a header and sometimes a body.

define html

Hypertext Markup Language, describing how content within the file is displayed.
This markup tells a web browser how to display text, images and other forms of multimedia on a webpage.

define http

Hypertext Transfer Protocol; HTTP is used to transfer data over the web. 
It is part of the Internet protocol and defines commands and services used for 
transmitting webpage data.


Define Front End

Client-side development producing HTML, CSS and JavaScript for websites or web
application so the user can see and interact. Front end development is that the 
tools and techniques used to create the front end of a website change constantly and so the developer needs to constantly be aware of how the field is developing.
